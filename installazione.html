Passo 1: Prima di iniziare l'installazione di Ubuntu, assicuratevi di eseguire il backup dei vostri dati importanti e di verificare i requisiti di sistema consigliati. Avrete bisogno di una chiavetta USB con almeno 12 GB di spazio disponibile.

Passo 2: Andate sul sito ufficiale di Ubuntu (https://ubuntu.com/) e scaricate l'ultima versione stabile del sistema operativo. Scegliete tra le versioni LTS (Long Term Support) o le versioni non LTS in base alle vostre preferenze.

Passo 3: Utilizzate un'applicazione come Rufus su Windows o Etcher su macOS e Linux per creare una chiavetta USB avviabile con l'immagine di Ubuntu appena scaricata. Inserite la chiavetta USB nel vostro computer.

Passo 4: Riavviate il vostro computer e selezionate la chiavetta USB come dispositivo di avvio. Seguite le istruzioni sullo schermo per avviare il processo di installazione di Ubuntu.

Passo 5: Durante l'installazione, vi verrà chiesto di configurare le partizioni del disco. Potete optare per l'installazione affiancata con un sistema operativo esistente o scegliere di utilizzare l'intero disco per Ubuntu.

Passo 6: Durante l'installazione, avrete la possibilità di personalizzare le impostazioni utente, come il nome del computer, il vostro nome utente e la password. Assicuratevi di ricordare queste informazioni.

Passo 7: Una volta completata l'installazione, riavviate il computer e rimuovete la chiavetta USB. Vi troverete ora nel sistema operativo Ubuntu appena installato.

Passo 8: Dopo l'accesso, aprite il terminale e aggiornate il sistema con il comando: sudo apt update && sudo apt upgrade
